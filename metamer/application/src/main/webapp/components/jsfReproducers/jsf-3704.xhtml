In order to get this working you have to do set max parameters count, before running this example. The required step are following:

  1. start the EAP
  2. go to JBOSS_HOME/bin
  3. run jboss-cli.sh (or bat)
  4. type connect
  5. type /system-property=org.apache.tomcat.util.http.Parameters.MAX_COUNT:add(value=14000)
  6. disconnect and stop the EAP
  7. Now you should be able to run this example.
